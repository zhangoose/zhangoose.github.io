So I use the debian-based linux distro called Crunchbang (that apparently not a lot of people know about). I use it because my netbook can’t do jackshit but Crunchbang (no, not even the windows starter  pack).

The sad part is that after 1.5 whole years of using this distro and after 2.5 whole years of using some sort of linux, I finally look up how to kill applications (like in Windows, we have task manager).

Turns out that if you hit Alt + F2, it’ll bring up the Run prompt <— I actually didn’t know that. I knew about the Alt + F3 thing, but not Alt + F2. Then, there’s this thing called xkill. So if you type that in, you can kill any application you want! It’s actually really friendly and easy to use. You just … click on the window whose application you want to kill. So simple. Why didn’t I know it before?
