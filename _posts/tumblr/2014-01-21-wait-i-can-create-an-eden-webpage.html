So a little birdie told me about how you can create a webpage using eden (for all you Rutgers people). This’ll be my short tutorial on how to do that just in case someone else asks me. Unfortunately, this also means that I will nolonger be direct linking my domain name (zhalex.net) to this Tumblr. But it’ll be linked to on my actual webpage! (Yesss, don’t have to buy server space).

This will be assuming that you have an eden.rutgers.edu account. I’m going to be accessing my files via SSH on bash. There are other tutorials on the eden site about how to access it using PuTTY or some other SFTP client. 

In bash, type in

ssh -l USERNAME eden.rutgers.edu

Replace USERNAME with your eden username (more commonly known as netid). 

You should then be prompted with a yes/no check to see whether or not you want to access eden.rutgers.edu. Type in yes, unless, y’know. After that, it should prompt you for your password. After typing that in, you should get to a directory layout that has

drwx------ 10 USERNAME students 4096 DATECREATED Maildir

Assuming you never touched the file management on your account.

So you could actually just access your site right now at eden.rutgers.edu/~USERNAME (make sure the ~ is there), but if you have no files on there, it’ll return with a Forbidden - you don’t have access to this  etc message. 

Here is when this became pretty helpful. In short, just create a directory called public_html and drop in html files that you want to be seen in your website.

You can access each of those files by going to eden.rutgers.edu/~USERNAME/FILENAME.

If there’s no file in your public_html directory called “index.html”, your site is going to list out all your files in that directory. By creating an “index.html”, you’re giving your site a customized-by-you page to go to when people type in eden.rutgers.edu/~USERNAME (which will be index.html). 

Hope that helps :D

PS- if you don’t like using consoles to do things, you can do file management using Weeble File Manager. It gives you like the barebones. I don’t use it only because it looks really ugly (actually, all of eden does), but that’s just me.
